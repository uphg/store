这两年没停过，一直在学习新东西，学习心得就写成了上面的教程。虽然看上去数量不少，但是下一代的互联网开发技术，我还是只学了很小一部分，像 PostCSS、GraphQL、Electron 这些感兴趣的东西，都没时间搞。

面对技术的高速发展和百花齐放，我有时也感到疲倦烦躁。但是，每当看到它们带来的生产力的飞跃，让你一个人快速搞定前后端的全部开发时，就觉得这终究还是一条正确的道路。
